# Examples

This directory contains example code demonstrations for the RealityLib library.

## Usage

Each example file is a self-contained code snippet that demonstrates a specific feature or functionality of RealityLib. 

**To run an example:**
1. Copy the desired example code
2. Paste it into your `main.c` file
3. Compile and run your project

## Available Examples

- **objectCreation** - Basic initialization and setup of floating colored cubes
- **cubeSliceGame** - A cube slicing game with combo and flip system

Refer to individual example files for detailed comments and documentation.
